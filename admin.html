🚀 Pubblicazione con un click (facoltativa)

Con un token GitHub il pulsante qui sotto carica da solo i file modificati sul sito: niente più scarica-e-trascina. Come crearlo: GitHub → foto profilo → Settings → Developer settings → Personal access tokens → Fine-grained → Generate new token → Repository access: solo questo repository → Permissions → Contents: Read and write → scadenza a tua scelta.

👁 Anteprima live delle modifiche (prima di pubblicare)

Mostra il sito vero con i dati che stai modificando, senza toccare quello online.

L'anteprima live funziona quando il pannello è aperto dal sito online (o da un server locale). Aperto come file, usa la pubblicazione o il doppio click su index.html dopo aver sostituito i file.

Come funziona (3 passi)

  1. Modifica quello che vuoi nelle schede qui sopra (prodotti, testi, contatti). Tutto avviene in sicurezza nel tuo browser: il sito online non cambia ancora.
  2. Scarica i file aggiornati con i pulsanti nella barra in basso (scarica solo quelli evidenziati come modificati).
  3. Caricali su GitHub col pulsante «↗ Carica su GitHub»: si apre la pagina giusta, trascini i file scaricati, premi Commit changes. In 1–2 minuti il sito è aggiornato.
